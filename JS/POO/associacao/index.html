Uma associação é um relacionamento entre classes que permite que uma instância de uma classe faça a outra realizar uma associação

    é como dizer para aplicação que um objeto de um tip está conectado, ou depende, de um objeto de outro tipo
    Exemplo

    Monte uma classe, 
    a exporte para outro arquivo, 
    no novo arquivo a importe.
    para fazer com que a classe principal deste arquivo receba a outra classe
    use a sua constante atribua a ela seus parametros
