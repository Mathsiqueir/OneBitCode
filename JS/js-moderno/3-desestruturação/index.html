Funcionalidade que permite criar Variáveis a partir
de propriedades de um objeto ou elemento do Array

no caso das propriedades as variáveis serão criadas com o mesmo nome da propriedade

tambem pode ser utilizadas de outra forma, como nos parâmetros
de um função

exemplos
const { name, email } = person
        const [x, y] = coordinates

        function hello({ name }) {
            console.log(`Hello, ${name}`)
        }
