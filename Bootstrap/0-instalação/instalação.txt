instalação CSS

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>




1. Acesse o link da nossa documentação: https://bootstrapdoconebitcode.netlify.app/ e copie a instalação CSS e coloque no head do HTML

2. Copie também a instalação de JavaScript e coloque antes do final do body. Agora o Bootstrap foi instalado por link, a parte CSS e também o JS.

3. Agora instalando através do npm você precisa copiar o código instalação completa, excluindo o $ na frente do código e colando no terminal, que pode ser do próprio VS Code.Dessa forma o Bootstrap também está instalado, só que agora dentro do seu projeto, fazendo com que você consiga manipular certas coisas utilizando o Sass.

4. O Bootstrap ele pode ser dividido em containers, você precisa apenas colocar uma classe de container em uma div, que dessa forma ele terá a propriedade container

5. Você pode clicar na navbar com o nome elementos para ver os elementos, vamos clicar em tipografia para usar um elemento dela.

6. Você pode ver que tem um parágrafo com uma classe de h1, que faz com que ele fique com um estilo de h1, e o container está fazendo a "limitação" dele.

7. Também é possível usar container-fluid para que ele fique ocupando a tela inteira, isso é muito útil e quase todos os nossos elementos com Bootstrap vão estar dentro de containers.

8.Para você navegar na nossa documentação, basta você acessar o link [https://bootstrapdoconebitcode.netlify.app/index.h...](https://bootstrapdoconebitcode.netlify.app/index.html) e vai para a página inicial, com isso você tem a barra de navegação que tem os elementos mais usados, de forma fácil para você!Entrando no elemento, você vai ver o título dele e logo abaixo, o link dele para a documentação oficial, para você explorar, pois todas as variações de todos os elementos não iriam caber aqui.
